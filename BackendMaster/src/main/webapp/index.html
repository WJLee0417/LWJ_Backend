📂 Project Architecture

StepUpBackend (Maven)
├── src/main/java
│ ├── com.test.dao     # MySQL Query Logic (Board, Member, Comment)
│ ├── com.test.dto     # Data Transfer Objects
│ ├── com.test.util    # DBUtil, PasswordUtil (SHA-256)
│ ├── com.test.filter   # Security & Encoding Filters
│ ├── com.test.listener # AppInitListener (Auto Admin Create)
│ ├── com.test.servlet  # Controllers (Servlet 6.0)
│ └── com.test.db      # Deprecated MockDB
├── src/main/resources/sql
│ └── init.sql       # Database Schema & Initial Data
└── src/main/webapp
    ├── index.html      # Mission Dashboard
    ├── login.jsp / join.jsp
    ├── board.jsp       # List View (Two-Track Paging)
    ├── boardDetail.jsp # Content & Comment View
    ├── boardWrite.jsp  # Create Form
    └── boardUpdate.jsp # Update Form

DATABASE

02. 연쇄 삭제 (On Delete Cascade)

SQL LOGIC

admin으로 공지를 쓰고, 일반 글을 10개 이상 써서 2페이지로 가보세요. 어떤 페이지에서도 📌공지사항은 항상 최상단에 고정되어야 합니다.

UI/UX

04. 역순 가상 번호 계산
